# 3PandaLabs — products

Entry point for every product. Each folder below is a **git submodule** pointing at that product's monorepo (`web/ + app/ + supabase/`) — click through to the real repo.

| Product | Repo | Live |
|---|---|---|
| ReceiptCash — receipt-based cashback rewards | [receiptcash](https://github.com/3pandalabs/receiptcash) | [receiptcash.3pandalabs.com](https://receiptcash.3pandalabs.com) |
| NRIGhar — property management for NRI landlords | [nrighar](https://github.com/3pandalabs/nrighar) | [nrighar.3pandalabs.com](https://nrighar.3pandalabs.com) |

Org-level repos (ops, brand, ideas, website) live under the sibling entry point: [org-level](https://github.com/3pandalabs/org-level) (private).

> Submodule pointers are navigational — they may lag the latest commit of each repo. Clone with `git clone --recurse-submodules` if you want the contents.
